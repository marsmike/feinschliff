# Code of Conduct

This project adopts the **Contributor Covenant 2.1** as its code of conduct.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

To report an incident, contact the maintainer at:

`mike@objektarium.de`

All reports are kept confidential. The maintainer is responsible for clarifying and enforcing standards of acceptable behavior, and will take appropriate corrective action in response to any reported incident.

## Scope

This Code of Conduct applies in all project spaces, and also when an individual is officially representing the project in public. Public spaces include GitHub issues, pull requests, discussions, and any project-affiliated communication channels.
