# Narrative Structures

Catalog of proven narrative approaches for explainer videos. Propose 2-3 of these when discussing narrative structure with the user.

## Problem → Solution

**Flow:** Introduce problem → Show pain/limitations → Present solution → Demonstrate how it works → Summarize benefits

**Best for:** Product explanations, tool introductions, process improvements

**Emotional arc:** Frustration → Recognition → Relief → Excitement

**Example scenes:**
1. Hook: "Ever struggled with X?"
2. Problem: Show the pain point visually
3. Solution: Introduce the tool/concept
4. How it works: Step-by-step walkthrough
5. Benefits: Summary of improvements

## Journey / Progression

**Flow:** Starting point → First step → Growth/evolution → Current state → Future potential

**Best for:** Technology evolution, learning paths, system architecture that builds up

**Emotional arc:** Curiosity → Understanding → Appreciation → Inspiration

**Example scenes:**
1. Origin: "It started with X..."
2. Evolution: Show how it grew/changed
3. Current: What it looks like today
4. Future: Where it's heading

## Before → After Comparison

**Flow:** Show the old way → Highlight limitations → Show the new way → Direct comparison → Clear winner

**Best for:** Migrations, upgrades, methodology comparisons

**Emotional arc:** Familiarity → Dissatisfaction → Discovery → Conviction

**Example scenes:**
1. Before: The current state
2. Pain points: What's wrong
3. After: The improved state
4. Comparison: Side-by-side
5. Verdict: Why it's better

## Question → Answer / FAQ

**Flow:** Ask compelling question → Explore wrong assumptions → Reveal answer → Explain why → Reinforce understanding

**Best for:** Concept explanations, myth-busting, educational content

**Emotional arc:** Curiosity → Surprise → Understanding → Confidence

**Example scenes:**
1. Question: "How does X actually work?"
2. Common misconception: What people think
3. Reality: How it actually works
4. Deep dive: The mechanism
5. Takeaway: Key insight

## Process Flow / How It Works

**Flow:** Overview → Step 1 → Step 2 → Step N → Complete picture

**Best for:** Technical processes, workflows, system architectures, algorithms

**Emotional arc:** Anticipation → Progressive clarity → Satisfaction

**Example scenes:**
1. Overview: High-level view
2. Steps: One scene per step, building up
3. Integration: How steps connect
4. Result: Complete flow in action

## Layered Reveal

**Flow:** Surface level → Deeper layer → Deepest layer → Full picture with all layers visible

**Best for:** Architectures, abstractions, systems with hidden complexity

**Emotional arc:** Simplicity → Growing complexity → Mastery

**Example scenes:**
1. Surface: What you see
2. Under the hood: First layer of complexity
3. Core: The fundamental mechanism
4. Full picture: All layers together
