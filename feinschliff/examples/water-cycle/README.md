| Field | Value |
|---|---|
| Frame | SCQA |
| Audience | Educated-adult explainer (NYT science / Crash Course tone) |
| Brand | `nord` |

## Download

The deck binaries are hosted on R2 to keep this repo lightweight — the plugin
install ships the renderer, not the rendered output.

- **[`deck.pdf`](https://assets.marsmike.com/feinschliff/examples/water-cycle/deck.pdf)** — rendered PDF for review (~3 MB)
- **[`deck.pptx`](https://assets.marsmike.com/feinschliff/examples/water-cycle/deck.pptx)** — editable PowerPoint / Keynote source (~16 MB)
- **[`speaker-book.pdf`](https://assets.marsmike.com/feinschliff/examples/water-cycle/speaker-book.pdf)** — annotated speaker handout: per-slide thumbnail + claim + speaker notes + role + audience fit (30 slide pages + 1 front-matter page, ~3 MB)

## What this deck demonstrates

Feinschliff's full surface mix in a single narrative deck:

- **26+ distinct layouts** across 30 slides — covers, editorial, data, strategic, diagrams, text.
- **7 image-slot layouts** authored alongside this deck — `photo-grid`, `kpi-photo`, `chart-photo`, `full-bleed-editorial`, `end-image`, `agenda-photo`, `photo-strip-four`. All ship as toolkit-shared and inherited by every brand pack.
- **18 photos** sourced from Unsplash + NASA (per-slide credits in [`ATTRIBUTION.md`](ATTRIBUTION.md)).
- **2 centerpiece diagrams** — landscape-style SVG water cycle (slide 7) + bespoke SVG "drawn to scale" reservoir cross-section (slide 5).
- **16 cited data slides** drawing from USGS, IPCC AR6, NOAA SOTC, ECMWF, FAO AQUASTAT, UN WWAP, WRI Aqueduct, NSIDC, and WGMS. Citations live in PPTX speaker notes so the deck doubles as a self-contained reference.
- **Speaker book** (handouts) generated by `feinschliff deck book` — pairs each slide thumbnail with its red-line role, claim sentence, audience-fit note, and the speaker notes the presenter walks through live.

## Story arc (SCQA)

| Act | Slides | Beat |
|---|---|---|
| Cover + open | 1–3 | Title · agenda (photo) · hook |
| Act 1 — Scale | 4–8 | Reservoirs, drawn to scale, the cycle as a system, Heraclitus closer |
| Act 2 — Stress | 9–18 | 350 years of measurement, intensification, regime shifts, risks, monitoring outcomes |
| Act 3 — Pathways | 19–26 | Where each drop goes — residence times, phase overlaps, urban loop, fate of a rainfall, water-use hierarchy, six exemplars worldwide |
| Act 4 — Response | 27–29 | Three Pyramid-Principle moves; three takeaways; four futures |
| Close | 30 | "Same water." |

## How it was built

```bash
# Deck
uv run feinschliff deck build .debug/water-cycle/content_plan.yaml \
    -o examples/water-cycle/deck.pptx
uv run feinschliff verify examples/water-cycle/deck.pptx --json

# Speaker book (handouts)
uv run feinschliff deck book .debug/water-cycle/content_plan.yaml \
    --design-brief .debug/water-cycle/design_brief.json \
    --pptx examples/water-cycle/deck.pptx \
    -o examples/water-cycle/speaker-book.pdf
```

Build outputs (`deck.pptx`, `deck.pdf`, `speaker-book.pdf`, thumbnails) are
hosted on R2 — see the **Download** links above. The repo itself ships only
the renderer + this README + [`ATTRIBUTION.md`](ATTRIBUTION.md).

Inputs (gitignored under `feinschliff/.debug/water-cycle/`):
- `content_plan.yaml` — 30-slide composer input + speaker notes
- `design_brief.json` — front-matter + per-slide role / claim / audience_fit (feeds the speaker book)
- `images/` — cached Unsplash / NASA photos
- `manifest.yaml` — image attribution metadata

## What this is NOT

- Not a research paper — citations live in speaker notes; on-slide numbers are rounded and qualified.
- Not a polemic — register is *explainer*, not advocacy. No "we must…" claims.
- Not the only canonical example — the per-layout taxonomy template at
  [`Feinschliff-Template.pdf`](https://assets.marsmike.com/feinschliff/examples/feinschliff/Feinschliff-Template.pdf)
  ships one slide per layout as a layout-completeness proof.
